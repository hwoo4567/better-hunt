# 아이템 드롭
플레이어가 죽었을 때 아이템을 일부만 드롭합니다.

# 명령어
"/betterHunt" 로 시작한다
## position
플레이어 좌표를 보여준다
* hide
* me
* nearest
* furthest

## timer
타이머 설정\
시간이 지나면 명령어를 실행했던 곳으로 귀환한다
* start <second>
* stop

## border
월드보더 설정
* set <size>

## info
모드 정보 보기
* recipes
